# Art generation prompts

Generated by `npm run artgen:prompts`. Make the art by hand in any image tool
(ChatGPT, Gemini, etc.), then **save each PNG to the exact path shown** and
refresh the app — images load automatically.

Tips:
- Do the **reference portrait** for each character first, then upload it as a
  reference when generating that character's outfit states so it stays the same
  character.
- Match the aspect ratio noted for each image.
- ✅ = the file already exists · ⬜ = still needed.
